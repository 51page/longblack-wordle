# 롱블랙 단어 맞추기 (LongBlack Word Game)

롱블랙 노트 제목을 맞추는 워들 스타일의 게임입니다. 매일 새로운 제목이 공개됩니다!

## 주요 기능
- **매일 새로운 문제**: 54개의 롱블랙 노트 제목 중 매일 하나씩 출제됩니다.
- **힌트 시스템**: 문제의 글자 수와 설명을 통해 힌트를 얻을 수 있습니다.
- **테스트 모드**: 새로고침 시 게임이 초기화되어 여러 번 테스트가 가능합니다. (정식 배포 시 변경 가능)
- **결과 공유**: 이모지를 통해 내 결과를 친구들에게 공유할 수 있습니다.

## 기술 스택
- HTML5, CSS3, Vanilla JavaScript

## 시작하기
`index.html` 파일을 브라우저에서 열면 바로 게임을 실행할 수 있습니다.
